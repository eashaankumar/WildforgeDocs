# Wildforge Engine

A modern C# game engine focused on Vulkan rendering, raytracing, compute shaders, UI, networking, and game simulation.

---

# Features

- Vulkan Renderer
- Custom Shader Language
- Hardware Raytracing
- Compute Shaders
- Raster Rendering
- Render Passes
- Rich Text Rendering
- UI Rendering
- Scene Management
- Networking
- Asset Pipelines
- ForgeShader Language
- Typed Entity Pools
- Unmanaged Fast Data Structures
- Job System

---

# Philosophy

Wildforge Engine is built around:

- Explicit APIs
- High performance
- Modern graphics
- Full source ownership
- Minimal hidden behavior
- Engine-first architecture

---

# Status

Wildforge Engine is currently under active development and powers the upcoming sandbox adventure game **Forgewild**.
